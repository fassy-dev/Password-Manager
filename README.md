# Менеджер паролей (Password Manager)

Простой и безопасный консольный менеджер паролей на Python с функцией генерации надежных комбинаций и защитой доступа через мастер-пароль.

## 🚀 Функционал

*   **Защита авторизации**: ограничение количества попыток ввода мастер-пароля с задержкой между попытками.
*   **Управление данными**: просмотр, добавление и удаление учетных записей.
*   **Генератор паролей**: автоматическое создание случайных паролей (буквы, цифры, спецсимволы).
*   **Локальное хранение**: данные сохраняются в структурированном виде в файл `passwords.json`.

## 📂 Структура проекта

```text
папка_проекта/
├── src/
│   └── fassydev/
│       └── manager_passwords/
│           ├── config.py          # Конфигурация (имя БД, мастер-пароль)
│           ├── storage.py         # Логика чтения и записи JSON-файла
│           ├── generator.py       # Генератор случайных паролей
│           └── auth.py            # Модуль аутентификации пользователя
├── main.py                        # Главный интерфейс и меню управления (точка входа)
└── README.md                      # Документация проекта
```

## 🛠 Требования

*   Python версии **3.7** или выше.
*   Используются только встроенные библиотеки (`json`, `os`, `random`, `time`, `string`, `getpass`), поэтому установка сторонних пакетов через `pip` не требуется.

## 💻 Запуск

Для запуска программы выполните команду из корневой папки проекта:

```bash
python main.py
```

## 🔐 Настройки по умолчанию

*   **Дефолтный мастер-пароль**: `1` (изменить можно в файле `src/fassydev/manager_passwords/config.py`).
*   **Количество попыток ввода**: `3`.
*   **Длина генерируемого пароля**: `8` символов.
